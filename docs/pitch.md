# 🎙️ kaiwa — デスクの会話を、取りこぼさない！

---

ZoomやTeamsのミーティングって、最近は録音も文字起こしも自動でやってくれますよね！ tl;dvやOtter.aiみたいな便利なツールもたくさんあって、オンライン会議の記録はもう困らない時代です。

**でも、デスクで突発的に始まる会話ってどうしてますか？**

隣の人が「ちょっといいですか？」って話しかけてきて、5分のつもりが気づいたら30分。来期の方針とか、あのバグの対応方針とか、結構大事なことが決まったりするんですよね！

で、翌日。

> 「昨日デスクで話してた件、結論なんでしたっけ？」

…誰も覚えてない。メモも取ってない。😇

**これ、めちゃくちゃあるあるじゃないですか！？**

オンラインの会議にはツールがあるのに、**リアルの会話には何もない。** 大事な決定って、意外と会議室じゃなくて廊下やデスクの横で生まれるのに…！

この「録れなかった会話」をなんとかしたくて、作りました！

---

## kaiwa の使い方、めちゃくちゃシンプルです！

**ホットキーを押すだけ！** あとは全部自動です！

![kaiwa ワークフロー](assets/images/workflow.png)

録音が始まって、もう1回押したら止まります。そこから先はkaiwaが全部やってくれます！

- 🎤 **文字起こし** — 誰が何を言ったか、全部テキストになります！
- 👥 **話者分離** — 「これAさん、これBさん」って自動で分けてくれます！
- 📝 **AI要約** — 要点・決定事項・TODOをまとめてくれます！

5分の会話ならたったの17秒で処理完了。めっちゃ早いです！🚀

---

## macOSの通知で進捗がわかります！

録音中も処理中も、macOSの通知でリアルタイムに状況がわかります！

![kaiwa 通知フロー](assets/images/notification-flow.png)

ホットキーを押したら録音開始 → もう一度押したら停止 → あとは通知が来るのを待つだけ！

---

## こんなMarkdownが自動で出てきます！

完了すると、こんな感じの見やすいMarkdownファイルが自動で保存されます👇

![kaiwa 出力プレビュー](assets/images/output-preview.png)

**要約・TODO・全文書き起こし**が全部入ったMarkdownファイル！これがホットキー1つで手に入ります！

---

## 「既存のアプリでよくない？」って思いますよね！

はい、Plaud NoteやClova Noteなど、いいサービスがあるのは知ってます！ でもですね…

**🔒 音声データ、クラウドに送って大丈夫ですか？**
クライアントとの話や社内の機密情報って、「クラウドに送信されます」だと使えないシーンありますよね！ kaiwaは文字起こしも話者分離も**全部Mac上で完結します！** 音声データは外に出ません！

**💸 毎月2,000円、地味に痛くないですか？**
kaiwaはOSSなので基本無料です！ AI要約を使っても**月66円くらい**（1日2時間×週5日の場合）。要約を使わなければ完全タダです！

**🔧 「ここだけ変えたいんだよな…」ができます！**
要約のプロンプトを変えたり、出力フォーマットをいじったり、保存先を変えたり。痒いところに手が届きます！

## セットアップはたったの10分！

```bash
git clone https://github.com/takayama-tigrit/kaiwa.git
cd kaiwa
./setup.sh
```

あとはHuggingFaceのトークンとAnthropic APIキーをMac Keychainに登録するだけです！

iPhoneのボイスメモで録音 → iCloud経由でMacに自動同期 → kaiwaが自動処理、っていう連携もできちゃいます！📱
